배포본 구성요소

JAVA용 표준보안API의 패키지를 받으면 아래와 같은 파일들로 구성되어있습니다.

구분 파일 설명
conf gpkiapi.conf 환경파일은 인증서 검증, 본인확인 시에 접속하고자 하는 "통합검증서버"의 위치를 확인하기 위해서 사용됩니다.
info gpkiapi_info gpkiapi 의 버전을 확인할 수 있는 실행 파일 입니다.
jar libgpkiapi_jni.jar JAVA 용 표준보안API 클래스 묶음 파일 입니다.
javadoc index.html JAVA용 표준보안API 사용을 위한 메뉴얼 입니다.
jtest java/
Cert.java, Cms.java, Crypto.java, Ivs.java, Main.java, Tsa.java		 제공하는 표준보안API를 이용한 각 모듈별 샘플 코드입니다.
class/
Cert.class, Crypto.class, Main.class, Tsa.class, Cms.class, Ivs.class, Sample/		 위의 샘플 코드를 컴파일한 결과 입니다. Sample 디렉토리에 있는 데이터는 샘플 코드가 구동되면서 필요한 데이터들 입니다.
lib gpkiapi.dll (libgpkiapi.so(sl,a)) C/C++용 표준보안API(gpkiapi)를 C/C++, JAVA에서 호출할 수 있도록 JNI를 구성한 모듈 입니다.
readme - 배포된 표준보안API의 버전과 구성 내용을 확인할 수 있는 파일입니다.
